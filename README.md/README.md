# Indian Painting Style Classification

## Overview
This project classifies Indian paintings into different traditional art styles using Machine Learning.  
The goal is to understand how visual patterns in artwork can be learned and classified by algorithms.

The project follows a complete ML pipeline:
data loading → preprocessing → model training → evaluation.

---

## Problem Statement
Given an image of an Indian painting, predict its art style  
(e.g., Madhubani, Warli, etc.).

This problem is relevant in:
- Digital art archives
- Cultural heritage preservation
- Art recommendation systems

---

## Dataset
- Image dataset organized into folders by painting style
- Each folder name represents a class label
